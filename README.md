# Business AI Playbook App

A lightweight web app that turns the provided AI SOP into a user-friendly, interactive guide.

## Features
- Guided 4-step AI deployment checklist with progress saved in browser localStorage.
- Searchable prompt library across departments.
- Master Prompt template and RCCF format ready to copy.
- Quick CTA link to the Elite Coaching Instagram.

## Run locally
```bash
python3 -m http.server 4173
```
Then open `http://localhost:4173`.
